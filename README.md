# Git-Resource
這是我收集了一些跟 Git 有關的資源，因為我原本都用Xcode內建的工具做Git版本控制，不過發現還是需要學一下Git的指令，所以上網找了一下看看有什麼教學資源，沒想到網路上資源超多，想說就整理一下，方便以後如果要找Git相關的資料就來這裡找，實際上大部份我自己都還沒看過......
